# Multi-Threaded Primality Benchmark
This project compares the performance of single-threaded vs. multi-threaded primality testing across **Java**, **Kotlin**, and **Go**.

## Prerequisites
- **Go**: 1.18+
- **Java**: JDK 11+
- **Kotlin**: 1.6+ (kotlinc)

## How to Run
1. Give execution permissions to the script:
    bash
   chmod +x run.sh
