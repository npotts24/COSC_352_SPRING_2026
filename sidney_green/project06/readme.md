# Baltimore City Homicide Analysis Dashboard

This project is a containerized web application that scrapes Baltimore City homicide data from 2023–2025 and provides an interactive dashboard for statistical analysis and visualization.


## 🚀 Getting Started

### Prerequisites
* Docker must be installed and running.
* If using GitHub Codespaces, Docker is pre-configured.

### Installation & Setup
1.  Navigate to the project directory:
    ```bash
    cd sidney_green/project06
    ```
2.  Ensure the scripts have execution permissions:
    ```bash
    chmod +x dashboard.sh
    ```

### Running the Application
Run the automated build and execution script:
```bash
./dashboard.sh
