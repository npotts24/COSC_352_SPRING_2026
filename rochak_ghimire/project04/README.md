#Project 4 - Multi Language Single and Multi Threaded Prime Counter


## Overview
This project  counts the total number of prime numbers present in a text file using java, kotline, and Go.

All of these languages implements:
- Single threaded prime counting
- Multi threaded prime counting 
- Efficient primality check

Outputs:

- Total no of primes present
- Execution time for single and multi threaded run times
- Speed ratio

##Requirements:

- macOS/ Linus
- java 17+
- kotlin
- Go
- Bash


##Running Requirements:
- Go isndide the "project04" folder
- Ensure run.sh is executable
