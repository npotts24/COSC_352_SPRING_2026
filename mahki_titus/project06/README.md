# Baltimore Police Department Homicide Dashboard

This project builds an interactive crime analysis dashboard using R Shiny for the Baltimore City Police Department.

The dashboard allows detectives, analysts, and commanders to explore homicide patterns interactively.

## Features

### Interactive Filters
Users can filter homicide data by:
- Year
- Victim age range
- Method of homicide

### Visualizations

1. Monthly Homicide Trends (Interactive Line Chart)
2. Homicide Method Breakdown (Bar Chart)
3. Geographic Map of Incidents
4. Searchable Data Table

### Summary Statistics

The dashboard dynamically updates key metrics:
- Total homicides
- Clearance rate
- Average victim age

### Data Pipeline

The dashboard loads homicide data from:

https://github.com/washingtonpost/data-homicides

The data is automatically cleaned and processed when the application starts.
